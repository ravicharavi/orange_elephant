## Instagram Feed Setup

To display Instagram posts, you have several options:

### Option 1: Manual Post URLs (Easiest)
1. Open Instagram posts from @palanworks
2. Click the three dots (â‹¯) on a post
3. Select "Copy link"
4. Add the URLs to the `instagramPostUrls` array in `script.js`

### Option 2: Instagram Basic Display API
1. Create a Facebook App at https://developers.facebook.com/
2. Add Instagram Basic Display product
3. Get Access Token
4. Update the script to fetch posts from the API

### Option 3: Third-Party Services
- [Juicer.io](https://www.juicer.io/)
- [SnapWidget](https://snapwidget.com/)
- [Elfsight Instagram Feed](https://elfsight.com/instagram-feed-instashow/)
